# Rust-File-CLI-tool
A command line tool in Rust which can do various operations on files.

to download
```
git clone https://github.com/MidhunManu/Rust-File-CLI-tool
```
read file 
```
file read <filename>
```

to read
```
file write <filename>
```

to create an empty file
```
file create <filename>
```
